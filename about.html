About CMS

College Committee Management System is a web- based project aims to faciliate committees work and its focuses on student activities committee. It is a new technical way to manage all committee related jobs. In a propose system students can get to know about committee, view their events and activities organised as well as their upcoming events. The main idea of this project is to maintain the college committee events information and organised the events and to send the student registration time through sums with verification code to the student using website.

Vision & Mission:

  • To enable the user to identify the committee he/she should join based on his/her interests.
  • To eliminate the use of various other methods for getting the information about the committees.
  • To help the existing committees manage their data (i.e., number of members, vacant positions, schedule of upcoming events, and many more) in an easier and simplified way.

Objectives

  • To enable the user to know what type of events are organized by the particular committee.
  • To enable the user to directly send an application for joining a particular committee.
